# Customer Ad Click Prediction Using Scikit-Learn

## Overview

This project predicts whether a customer will click on an online advertisement using demographic and purchase history data. A Logistic Regression model is trained using Scikit-Learn to classify customer behavior.

## Objective

To build a machine learning classification model that predicts ad-click behavior based on:

* Age
* Gender
* Past Purchases

## Dataset

The dataset contains 200 customer records with the following attributes:

| Column        | Description                       |
| ------------- | --------------------------------- |
| Age           | Customer age                      |
| Gender        | Male/Female                       |
| PastPurchases | Number of previous purchases      |
| ClickedAd     | Target variable (0 = No, 1 = Yes) |

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Google Colab

## Machine Learning Workflow

1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Logistic Regression Training
6. Prediction
7. Model Evaluation

## Evaluation Metrics

The model is evaluated using:

* Accuracy
* Precision
* Recall
* F1 Score

## Results

The Logistic Regression model successfully predicts customer ad-click behavior using demographic and purchase-related features.
